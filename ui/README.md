# 🚀 Twitch-Channel-Points-Miner-v2 (Web Interface)

**Twitch-Channel-Points-Miner-v2** — это современная, высокотехнологичная панель управления (Dashboard) для популярного Python-майнера баллов канала. 

### ⚠️ ВАЖНОЕ ПРИМЕЧАНИЕ
Данный репозиторий содержит **ТОЛЬКО веб-интерфейс** и вспомогательный прокси-сервер. Для полноценной работы требуется наличие запущенного основного скрипта майнера (Python core), с которым эта панель взаимодействует через API и конфигурационные файлы.

---

## 📋 Описание проекта

Этот интерфейс заменяет стандартную консоль Python на интуитивно понятный, быстрый и красивый UI, позволяя управлять десятками аккаунтов-ботов с любого устройства.

### Ключевые функции Web-части:
*   **📡 Мониторинг в реальном времени**: Отслеживание каждого балланса и статуса просмотра без перезагрузки страницы.
*   **🎭 Массовое управление**: Активация наград, запуск или остановка ботов одной кнопкой сразу для всех сессий.
*   **🛠 Глубокая настройка**: Изменение параметров стримеров, приоритетов и лимитов через визуальные формы.
*   **📈 Статистика ресурсов**: Мониторинг нагрузки на процессор и память сервера, где запущен майнер.

---

## 🏗 Технологический стек

*   **Frontend**: React 18, TypeScript, Vite.
*   **Анимации**: Framer Motion (библиотека `motion/react`) для плавных переходов и интерактивности.
*   **Стилизация**: Tailwind CSS (современный dark-mode дизайн).
*   **Backend (Proxy)**: Express.js (используется для связи фронтенда с файловой системой и API майнера).
*   **Иконки**: Lucide React.

## 📁 Подробный разбор интерфейса и функционала

### 1. 🏠 Обзор (Dashboard)
Центральный хаб статистики.
*   **Метрики**: Общее количество собранных баллов, количество активных сессий, полученные Drops за сегодня.
*   **Графики**: Визуализация доходности за последние 24 часа.

### 2. 📺 Активные стримы
Список каналов, которые боты смотрят прямо сейчас.
*   **Карточки стримов**: Показывают статус трансляции, количество ботов на канале и время до следующего бонуса (+50 баллов).
*   **Прогресс-бары**: Визуальное отображение накопления баллов за время просмотра (Watch Streaks).

### 3. 🎥 Стримеры (Менеджер каналов)
Управление списком целевых каналов.
*   **Добавить стримера**: Модальное окно для ввода логина канала. Настройки включают:
    *   *Авто-сбор*: Нужно ли собирать бонусные баллы.
    *   *Приоритет*: Очередность выбора канала ботами.
*   **Удаление**: Быстрое исключение канала из списка отслеживания.

### 4. 👥 Аккаунты (Управление ботами)
Главный раздел для контроля сессий.
*   **Кнопка "Запустить"**: 
    *   Открывает модальное окно.
    *   Позволяет выбрать конкретных стримеров или "Всех".
    *   Позволяет выбрать конкретные оффлайн-аккаунты для инициализации.
*   **Кнопка "Остановить"**: 
    *   Открывает модальное окно со списком всех активных (Running) ботов.
    *   Позволяет точечно или массово завершить сессии.
*   **Добавить аккаунт**: Загрузка новых `.pkl` файлов сессий через Drag-and-Drop или выбор файла.
*   **Карточки ботов**: Отображают имя, файл сессии, текущий статус (Active/Offline) и быстрый доступ к токену.

### 5. 💎 Активация наград (Баллы)
Глобальное действие в боковом меню.
*   Позволяет выбрать стримера и бота.
*   **Умные поля**: Если выбранная награда требует ввода текста (например, "Выделить сообщение"), автоматически появляется поле для ввода.
*   **Массовость**: Возможность отправить запрос на активацию сразу для всех сессий на канале.

### 6. 📜 Консоль логов
Интерактивный терминал.
*   Выводит системные события, ошибки API Twitch и отчеты о получении баллов.
*   Поддерживает фильтрацию по уровню важности (Info/Warning/Error).

### 7. ⚙️ Система
Мониторинг сервера.
*   Статус Docker-контейнера.
*   Версия Node.js и Python-ядра.
*   Использование CPU/RAM в реальном времени.

## 🌐 API Маршруты (Backend Proxy)

Сервер `server.ts` обрабатывает следующие запросы:

| Метод | Путь | Описание |
| :--- | :--- | :--- |
| `GET` | `/api/health` | Проверка доступности прокси-сервера. |
| `GET` | `/api/system` | Получение данных о загрузке системы (CPU/RAM). |
| `POST` | `/api/activate-reward` | Отправка команды майнеру на трату баллов. |
